# How to Share This Package

When sharing the Gong MCP Server with others, include these **4 files**:

## Files to Share

1. **`gong_mcp-0.1.0-py3-none-any.whl`** - The package file (from `dist/`)
2. **`install.bat`** - Installation script for Windows (this folder)
3. **`install.sh`** - Installation script for Mac/Linux (this folder)
4. **`INSTALL.md`** - Simple installation instructions (this folder)

## How to Share

### Option 1: Zip File (Easiest)

1. Copy the `.whl` file from `dist/` into this `install/` folder
2. Zip the entire `install/` folder
3. Share the zip file

**Name it something like:** `gong-mcp-server-v0.1.0.zip`

### Option 2: Individual Files

Share all 4 files together (via email, file sharing service, etc.)

Make sure recipients get all 4 files in the same location.

## What Recipients Do

1. Extract/unzip the files (if you sent a zip)
2. Follow the instructions in `INSTALL.md`
3. Double-click `install.bat` (Windows) or `install.sh` (Mac/Linux)
4. Done!

## Quick Checklist

Before sharing, make sure:
- [ ] The `.whl` file is included
- [ ] `install.bat` is included
- [ ] `install.sh` is included
- [ ] `INSTALL.md` is included
- [ ] All files are in the same folder/zip

That's it! Recipients have everything they need.
